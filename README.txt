Student Portal System - ICT 9 Project

Sample login:
Username: student
Password: ict9

Whats in it:
- Login checks username and password.
- Sidebar links open real pages.
- Search filters visible cards.
- Dashboard progress, quote, calendar, and guessing game work.
- Notebook subject cards open embedded PDF lessons from the lessons folder.
- Tasks can be marked done and dashboard progress updates.
- Profile saves username, password, display name, and picture using localStorage.
